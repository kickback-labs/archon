## GCP Compute Engine

### Description
Google Compute Engine is GCP's Infrastructure-as-a-Service (IaaS) offering that provides scalable, high-performance virtual machines running in Google's data centers. VMs can be configured with a wide range of machine types — including general-purpose (N-series, E-series), compute-optimized (C-series), memory-optimized (M-series), and accelerator-optimized (A-series with NVIDIA GPUs and TPUs) — and run on Linux or Windows. Compute Engine supports Spot VMs for fault-tolerant workloads at significantly reduced cost, as well as committed-use discounts and sustained-use discounts that automatically apply without upfront commitment.

### Use Cases
* General-purpose application hosting (e.g., web servers, APIs, and backend services)
* High-performance computing and batch processing (e.g., HPC clusters using MPI workloads with H4D)
* GPU/TPU-accelerated AI and ML workloads (e.g., model training on A3 instances)
* Lift-and-shift migration of on-premises VMs to the cloud
* Custom OS and kernel configurations requiring full control over the VM environment
