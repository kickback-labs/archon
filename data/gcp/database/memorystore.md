## GCP Memorystore

### Description
Memorystore is a fully managed in-memory database service supporting Valkey, Redis Cluster, Redis (standalone), and Memcached, delivering sub-millisecond data access without the operational overhead of self-managed caching infrastructure. It automates provisioning, replication, failover, and patching, and integrates with Cloud Monitoring and VPC for secure private connectivity via Private Service Connect. Memorystore for Valkey and Redis Cluster support zero-downtime horizontal scaling up to 250 nodes and terabytes of keyspace with a 99.99% availability SLA.

### Use Cases
* Application caching to reduce database load and accelerate response times for high-traffic web services
* Session management for stateless application tiers requiring fast, consistent session retrieval
* Real-time leaderboards, counters, and rate limiting for gaming and API platforms
* Vector search for generative AI applications requiring ultra-low-latency semantic retrieval
