## GCP Firestore

### Description
Firestore is a fully managed, serverless NoSQL document database that scales automatically to virtually unlimited capacity with up to 99.999% availability. It supports both a native document model and a MongoDB-compatible API (Enterprise edition), making it suitable for web, mobile, IoT, and AI applications. Built-in real-time synchronization, offline support, and client SDKs for iOS, Android, and web enable responsive multi-user apps without custom backend code.

### Use Cases
* Mobile and web app backends requiring real-time data sync and offline access (e.g., live chat, collaborative editing)
* Gaming backends for persistent user profiles, game state, leaderboards, and in-game inventories
* Generative AI applications using Firestore's vector search for RAG pipelines and semantic retrieval
* Content management systems serving dynamic, metadata-rich content to web and mobile apps
