## GCP Google Distributed Cloud Connected

### Description
Google Distributed Cloud Connected (GDC Connected) is a fully managed hardware and software solution that brings Google Cloud's infrastructure, AI capabilities, and Kubernetes-based developer experience to on-premises data centers and edge locations while maintaining connectivity to Google Cloud. Unlike the air-gapped variant, the connected edition maintains a persistent link to Google Cloud for management, billing, and access to Google Cloud APIs and services. It runs on ruggedized, Google-supplied hardware and is optimized for latency-sensitive and data-locality use cases in industries such as retail, manufacturing, and telecommunications. GDC Connected supports Google Gemini AI models on-premises, enabling inference at the edge without sending data to the public cloud. It is available in 25+ countries and supports a Kubernetes-based workflow, an active ecosystem of partners, and Google Cloud Ready–validated third-party software. Pricing starts at approximately $415 per node per month with multi-year discounts available.

### Use Cases
* Low-latency AI inference at the edge (e.g., real-time anomaly detection on factory floors)
* Retail store modernization (e.g., store analytics, fast checkout, predictive analytics across thousands of locations)
* On-premises generative AI workloads that require data residency but still need Google Cloud connectivity
* 5G network function deployment for telecommunications providers (e.g., cloud-native network modernization)
* Regulated industries requiring on-premises compute with cloud-managed control plane (e.g., financial services, healthcare)
* Survivability scenarios where edge workloads must continue operating during intermittent connectivity
