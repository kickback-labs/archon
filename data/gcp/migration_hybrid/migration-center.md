## GCP Migration Center

### Description
Google Cloud Migration Center is a unified, fully managed migration platform that accelerates end-to-end cloud migration journeys from on-premises environments or other clouds to Google Cloud. It consolidates discovery, assessment, and planning into a single interface, providing automated asset discovery of existing infrastructure, total cost of ownership (TCO) reporting, and cloud spend estimation before committing to migration. Migration Center integrates with a variety of migration tooling for different scenarios—VM migration, container migration, mainframe modernization, and database migration—and surfaces actionable recommendations for right-sizing on Google Cloud. It connects to tools such as Migrate to Virtual Machines, Migrate to Containers, and Database Migration Service to create a coordinated migration workflow. Migration Center is free to use for the core assessment and planning features, with costs applying to the underlying Google Cloud resources consumed during migration.

### Use Cases
* Pre-migration discovery and inventory of on-premises servers, VMs, and databases (e.g., scanning VMware environments with collector agents)
* Cloud cost estimation and TCO modeling before committing to migration (e.g., right-sizing estimates for Compute Engine)
* Migration planning and tracking across multiple workload types (e.g., VMs, databases, containers) in a single dashboard
* Identifying workloads suitable for lift-and-shift versus re-platforming versus modernization
* Accelerating migrations from other clouds (e.g., AWS, Azure) to Google Cloud with cross-cloud discovery support
* Generating offline assessments and TCO reports for executive sign-off on migration business cases
