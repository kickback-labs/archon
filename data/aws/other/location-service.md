## Amazon Location Service

### Description
Amazon Location Service is a fully managed geospatial service that enables developers to add maps, places search, routing, real-time device tracking, and geofencing capabilities to applications without building or maintaining geospatial infrastructure. It provides high-quality map tiles and satellite imagery from data providers including Esri and HERE, and exposes APIs for geocoding, reverse geocoding, points-of-interest (POI) search, and autocomplete address entry. The routing API supports turn-by-turn directions, real-time traffic, truck restrictions, and multi-stop route optimization. Tracker and Geofence APIs enable real-time asset tracking with EventBridge integration for triggering automated actions on location events. Amazon Location Service integrates natively with AWS services including Lambda, EventBridge, and Cognito, and offers a free tier for getting started.

### Use Cases
* Fleet and asset tracking (e.g., real-time GPS monitoring of delivery vehicles, field equipment, or personnel)
* Geofence-triggered automation (e.g., triggering alerts when assets enter or exit defined zones via EventBridge)
* Location-based mobile and web applications (e.g., store locator, nearby POI search, interactive map rendering)
* Delivery route optimization (e.g., calculating optimal multi-stop routes considering traffic, time windows, and vehicle restrictions)
* Address standardization and geocoding (e.g., cleaning and normalizing customer address data for CRM or logistics systems)
* Supply chain visibility and traceability (e.g., tracking inventory movement across warehouse and fulfillment networks)
