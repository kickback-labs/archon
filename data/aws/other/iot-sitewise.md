## AWS IoT SiteWise

### Description
AWS IoT SiteWise is a managed service for collecting, organizing, and analyzing industrial equipment data at scale, enabling data-driven operations across manufacturing and industrial facilities. It connects to industrial data sources — including OPC-UA servers, SCADA systems, historians, and on-premises data servers — and ingests data into a structured asset model hierarchy organized at the enterprise, site, area, and machine level. SiteWise computes common industrial performance metrics such as OEE (Overall Equipment Effectiveness), uptime, and throughput automatically, and provides real-time dashboards, alarm management, and ML-based anomaly detection via integration with Amazon Lookout for Equipment. The service supports both cloud-based and edge deployments through SiteWise Edge, enabling local data processing and visualization in facilities with limited or intermittent cloud connectivity. It was recognized as a Leader in the 2024 Gartner Magic Quadrant for Global Industrial IoT Platforms.

### Use Cases
* Manufacturing OEE monitoring (e.g., tracking equipment efficiency across multiple production lines and shifts)
* Predictive maintenance for industrial assets (e.g., detecting early failure signals in motors, compressors, and conveyors)
* Remote asset monitoring across facilities (e.g., centralizing visibility of distributed plants and equipment in a single dashboard)
* Industrial data historian modernization (e.g., replacing legacy on-premises historians with a scalable cloud-native time-series store)
* Edge data processing and local visualization (e.g., running SiteWise Edge in a facility without reliable internet connectivity)
* Integration with industrial analytics pipelines (e.g., exporting SiteWise data to Amazon S3 and Athena for advanced analysis)
