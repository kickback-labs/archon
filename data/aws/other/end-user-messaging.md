## AWS End User Messaging

### Description
AWS End User Messaging is a managed messaging service that enables developers to integrate scalable, reliable multi-channel messaging capabilities — including SMS, MMS, WhatsApp, push notifications, and text-to-voice — directly into applications. It supports both one-way notifications and two-way conversational messaging, and includes OTP (one-time password) verification APIs for account security and multi-factor authentication workflows. AWS End User Messaging succeeded Amazon Pinpoint (which reached end-of-support on October 30, 2026) and consolidates the messaging channel APIs previously spread across Pinpoint and SNS Mobile Push. The service provides a global delivery infrastructure with regulatory compliance support across numerous countries, and integrates natively with other AWS services for workflow automation and analytics.

### Use Cases
* OTP and multi-factor authentication delivery (e.g., sending time-sensitive verification codes via SMS or WhatsApp during login)
* Transactional and alert notifications (e.g., appointment reminders, fraud alerts, shipping updates via SMS or push)
* Two-way conversational messaging (e.g., customer support via WhatsApp or SMS with bi-directional message exchange)
* Promotional and marketing message delivery (e.g., limited-time offer campaigns sent to opted-in subscribers)
* Mobile push notification delivery (e.g., in-app engagement messages to iOS, Android, and web browsers)
* Multi-channel messaging strategy (e.g., reaching end-users via their preferred channel with a single unified API)
