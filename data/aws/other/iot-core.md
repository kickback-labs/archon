## AWS IoT Core

### Description
AWS IoT Core is a fully managed cloud service that enables billions of IoT devices to securely connect, communicate, and interact with AWS cloud applications and other devices. It supports MQTT (3.1, 3.1.1, and MQTT 5), HTTPS, and LoRaWAN protocols, providing a managed MQTT message broker that handles trillions of messages with persistent connections and advanced message retention. Device connections are secured with mutual TLS authentication and end-to-end encryption, and a built-in rules engine filters, transforms, and routes device data to AWS services such as Lambda, S3, DynamoDB, and Kinesis.

### Use Cases
* Industrial IoT monitoring and predictive maintenance (e.g., collecting sensor data from factory equipment)
* Connected consumer products (e.g., home automation, smart appliances, security systems)
* Automotive and fleet connectivity (e.g., connected and electric vehicle telemetry)
* Smart city and public safety applications (e.g., traffic monitoring, environmental sensing)
