## Amazon Managed Blockchain (AMB)

### Description
Amazon Managed Blockchain (AMB) is a fully managed service for building resilient Web3 applications on both public and private blockchains without deploying or operating specialized blockchain infrastructure. It offers two main capabilities: **AMB Access**, which provides instant serverless or dedicated node access to public blockchains (Ethereum, Polygon, Bitcoin) and private blockchains (Hyperledger Fabric); and **AMB Query**, which exposes developer-friendly REST APIs for accessing standardized real-time and historical blockchain data across multiple networks without ETL pipelines. AMB eliminates the operational burden of managing blockchain node software, networking, and storage, and scales securely for both institutional-grade and consumer application builds. Pricing is usage-based with predictable API call costs.

### Use Cases
* Token-gated access and NFT verification (e.g., verifying event ticket NFT ownership via AMB Query to grant access)
* Digital asset wallet development (e.g., building multichain wallets with transaction history APIs and fully managed public blockchain nodes)
* Decentralized finance (DeFi) application backends (e.g., reading real-time on-chain state and historical events from Ethereum or Polygon)
* Enterprise private blockchain networks (e.g., building a supply chain provenance system on Hyperledger Fabric with managed nodes)
* Blockchain data indexing and analytics (e.g., querying standardized token balances and transaction logs without custom indexers)
* Web3 application development (e.g., powering dApps with reliable, high-availability RPC endpoints to Ethereum mainnet or testnets)
* Regulated asset tokenization (e.g., issuing and tracking tokenized securities or loyalty points on a permissioned Fabric network)
