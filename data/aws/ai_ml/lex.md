## AWS Lex

### Description
Amazon Lex is a fully managed conversational AI service for building voice and text chatbots, powered by the same deep learning technology as Alexa. It provides automatic speech recognition (ASR) and natural language understanding (NLU) to interpret user intent and manage multi-turn conversations. Lex supports generative AI-powered bot design, including a natural language bot designer that generates a baseline bot from a description and a Visual Conversation Builder (VCB) for collaborative design. Bots can be deployed across web and mobile applications, and integrated with messaging platforms (Facebook Messenger, Slack, Twilio SMS) and Amazon Connect for omnichannel contact center scenarios. Pricing is per API request with no upfront commitments.

### Use Cases
* Self-service IVR and chat for customer support: handle FAQs, account inquiries, and transactions without human agents (e.g., bank balance checks, flight status)
* Intelligent call routing: understand caller intent and transfer to the appropriate contact center agent with context
* Enterprise productivity bots: respond to HR, IT helpdesk, or finance queries via Slack or Microsoft Teams
* End-to-end transactional workflows: allow users to book appointments, raise tickets, or submit applications through natural language
* Omnichannel engagement: deploy the same bot logic across voice, chat, and SMS from a single Lex console
* Augmenting applications with AI chat: add conversational search and task completion to existing mobile and web apps
* Contact center agent assist: provide real-time recommendations to agents handling live calls via Amazon Connect integration
