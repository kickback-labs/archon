## AWS Kendra

### Description
Amazon Kendra is a fully managed intelligent enterprise search service powered by machine learning that enables employees and customers to find accurate answers from large, distributed content repositories using natural language queries. Unlike keyword-based search, Kendra uses NLP to understand query intent and retrieve precise answers — including direct passage extracts, FAQ answers, and ranked document results — from over 40 built-in connectors to sources such as SharePoint, Confluence, S3, ServiceNow, and Salesforce. It supports retrieval-augmented generation (RAG) workflows, allowing it to act as the retrieval layer for LLM-powered conversational experiences built with Amazon Bedrock. Results can be fine-tuned based on content attributes, user feedback, freshness, and access control.

### Use Cases
* Employee productivity portals: surface accurate answers to internal HR, IT, and policy questions across intranets and wikis
* Contact center self-service: power chatbots (via Amazon Lex) that retrieve accurate answers from knowledge bases to reduce agent volume
* RAG-based generative AI: use Kendra as the retrieval backend for LLM-powered Q&A applications built on Amazon Bedrock
* SaaS in-app search: embed ML-powered search into customer-facing products for faster information discovery
* Legal and compliance: search across regulatory documents, contracts, and case law with natural language precision
* Unified enterprise search: aggregate and query content from SharePoint, Confluence, S3, and other repositories through a single interface
* Developer documentation portals: help engineers find relevant code examples and API references through intelligent search
