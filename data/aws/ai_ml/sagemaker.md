## AWS SageMaker AI

### Description
Amazon SageMaker AI (formerly Amazon SageMaker) is a fully managed ML platform that provides purpose-built tools for the entire machine learning lifecycle — from data preparation and model training to deployment and monitoring. It supports building, training, and deploying both traditional ML models and foundation models (FMs) on fully managed, high-performance infrastructure. SageMaker integrates with Amazon Bedrock for generative AI and with SageMaker Unified Studio, which provides a single development environment spanning analytics and AI workloads.

### Use Cases
* Train and fine-tune large language models and foundation models (e.g., fine-tuning Llama on domain-specific data)
* Build and deploy custom ML models with managed infrastructure (e.g., fraud detection, demand forecasting)
* Run distributed training jobs on GPU clusters with SageMaker HyperPod
* Manage the full MLOps lifecycle including experiment tracking, model registry, and CI/CD pipelines
* Accelerate development with pre-built algorithms and one-click deployment via SageMaker JumpStart
