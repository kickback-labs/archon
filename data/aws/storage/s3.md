## AWS Simple Storage Service (S3)

### Description
Amazon Simple Storage Service (Amazon S3) is a fully managed object storage service offering industry-leading scalability, durability, and performance. S3 stores data as objects within buckets and is designed to deliver 99.999999999% (11 nines) durability and 99.99% availability by default, backed by the strongest SLAs in the cloud. It supports virtually unlimited storage capacity — scaling to exabytes — with no provisioning required, and charges only for what you use. S3 is private and encrypted by default, with granular access control via bucket policies, IAM policies, Access Points, and ACLs.

### Use Cases
* Data lake storage for analytics, AI/ML, and HPC workloads (e.g., feeding Amazon Athena or SageMaker)
* Backup and disaster recovery for databases, file systems, and application data
* Static website and media asset hosting with optional CloudFront CDN integration
* Long-term archival at lowest cost using S3 Glacier and S3 Glacier Deep Archive storage classes
