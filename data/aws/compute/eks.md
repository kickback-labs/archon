## AWS Elastic Kubernetes Service (EKS)

### Description
Amazon Elastic Kubernetes Service (Amazon EKS) is a fully managed Kubernetes service that enables teams of any size to build, run, and scale production-ready containerized applications on AWS, on-premises, and at the edge. EKS Auto Mode fully automates cluster infrastructure management for compute, storage, and networking with a single click, eliminating manual cluster operations. It integrates with the breadth of AWS services while remaining fully conformant with upstream Kubernetes, supporting GPU workloads, multi-cluster federation, and hybrid deployments via EKS Hybrid Nodes, AWS Outposts, and EKS Anywhere.

### Use Cases
* Deploying generative AI and ML inference at scale (e.g., distributed training and LLM serving on GPU node groups)
* Building internal developer platforms with self-service Kubernetes environments
* Scalable batch and big data processing (e.g., Apache Spark on EKS)
* Hybrid and edge application deployments (e.g., EKS Hybrid Nodes connecting on-premises servers)
* Migrating and modernizing microservices from on-premises Kubernetes clusters
