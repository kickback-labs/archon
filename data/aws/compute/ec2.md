## AWS Elastic Compute Cloud (EC2)

### Description
Amazon Elastic Compute Cloud (Amazon EC2) provides secure and resizable compute capacity in the cloud as virtual machines (instances). It offers the broadest and deepest compute platform with over 1,000 instance types, supporting Intel, AMD, and Arm processors (including AWS Graviton), and enables full control over the operating system, networking, and storage configuration. EC2 instances can be purchased on-demand, as Reserved Instances, or as Spot Instances for cost optimization, and carry a 99.99% SLA. Security is built into the foundation via the AWS Nitro System.

### Use Cases
* General-purpose web and application servers (e.g., hosting multi-tier web apps on EC2 Auto Scaling groups)
* High-performance computing and scientific simulations (e.g., HPC clusters using compute-optimized instances)
* Machine learning training and inference (e.g., GPU instances for deep learning workloads)
* Windows and SAP enterprise workloads (e.g., lift-and-shift of on-premises VMs)
* macOS development and CI/CD (e.g., on-demand Mac instances for Apple platform builds)
