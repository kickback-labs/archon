## Azure Machine Learning

### Description
Azure Machine Learning is an enterprise-grade, end-to-end managed platform for the full machine learning lifecycle, from data preparation and model training through deployment and monitoring. It supports both traditional ML and generative AI workflows, providing a unified studio for data scientists and ML engineers. The service integrates with Azure AI Foundry, enabling prompt flow authoring and LLM fine-tuning alongside classical model development. It includes AutoML, a managed feature store, MLOps pipelines with CI/CD, and a model catalog with foundation models from OpenAI, Meta, Hugging Face, and Cohere.

### Use Cases
* Custom ML model training (e.g., classification, regression, computer vision, NLP on tabular or unstructured data)
* Generative AI prompt engineering and LLM fine-tuning via prompt flow
* MLOps and CI/CD pipelines for reproducible, automated model workflows
* AutoML for rapid no-code model development on tabular data
* Responsible AI — fairness assessment, interpretability dashboards, and harm mitigation
