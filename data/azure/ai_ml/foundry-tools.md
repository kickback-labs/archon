## Azure Foundry Tools

### Description
Foundry Tools is Microsoft's unified collection of prebuilt, task-optimized AI capabilities available within Microsoft Foundry (formerly Azure AI Services). It bundles modular, production-ready tools — including Azure Language, Azure Speech, Azure Vision, Azure Translator, Azure Content Understanding, and content safety controls — accessible via a single, consistent API surface and manageable through the Foundry portal. Foundry Tools allows developers to enrich AI agents and applications with OCR, speech recognition and synthesis, NLP, computer vision, translation, and document analysis without requiring machine learning expertise, and to deploy these capabilities in the cloud or at the edge via containers. The service is consumption-based with pay-per-use pricing on individual tools, all governed by Foundry's organization-wide security, access control, and observability controls.

### Use Cases
* Adding NLP, speech, vision, or translation capabilities to AI agents built with Foundry Agent Service (e.g., voice-enabled customer service agents)
* Enriching agentic RAG pipelines with OCR and content extraction from PDFs, images, and audio
* Embedding task-specific AI preprocessing into workflows before calling larger LLMs (e.g., PII redaction, summarization, entity extraction)
* Deploying AI capabilities at the edge or in air-gapped environments via containerized tool deployments
* Multi-service applications that combine several cognitive capabilities under a single Azure resource and billing scope
