## Azure Microsoft Entra ID

### Description
Microsoft Entra ID (formerly Azure Active Directory) is Azure's cloud-based identity and access management service. It provides authentication, single sign-on (SSO), and authorization for users, devices, and applications across cloud and on-premises environments. Every Azure subscription and Microsoft 365 tenant is backed by an Entra ID directory. It supports modern protocols (OAuth 2.0, OpenID Connect, SAML) and enables Zero Trust security through risk-based Conditional Access policies.

### Use Cases
* Centralized identity and SSO for cloud and on-premises apps (e.g., enabling employees to sign in once to access Microsoft 365, Salesforce, and custom apps)
* Multi-factor authentication (MFA) and Conditional Access enforcement based on user risk, device compliance, or location
* Application identity and workload authentication (e.g., service principals and managed identities for Azure resources)
* B2B collaboration by inviting external users as guests with scoped access to internal resources
* Customer identity and access management (CIAM) for consumer-facing applications via External ID
