## Azure Migrate

### Description
Azure Migrate is a free, unified migration hub that covers every phase of a cloud migration journey — discovery, assessment, planning, and execution — for servers, databases, web applications, virtual desktops, and large-scale offline data transfers. A lightweight virtual appliance deployed in the on-premises environment continuously collects configuration and performance data, which feeds into business case analysis (TCO comparison, year-over-year cash flow, right-sizing) and Azure readiness assessments for VMs, SQL Server instances, and web apps. Agentless and agent-based migration engines handle VMware VMs, Hyper-V VMs, physical servers, and VMs from other public clouds with minimal downtime, and the hub integrates partner tools (Carbonite, Zerto, Turbonomic, etc.) for scenarios requiring specialized capabilities.

### Use Cases
* Discovering and inventorying on-premises VMware, Hyper-V, and physical server estates using the agentless Azure Migrate appliance as a starting point for migration planning
* Building a migration business case that quantifies on-premises TCO versus Azure costs and models the switch from CapEx to OpEx
* Assessing SQL Server instances for migration readiness and right-sizing to Azure SQL Database, Azure SQL Managed Instance, or SQL Server on Azure VMs
* Executing agentless lift-and-shift migrations of VMware VMs to Azure with near-zero downtime using continuous replication
* Migrating ASP.NET and PHP web applications to Azure App Service at scale using the integrated App Service Migration Assistant
