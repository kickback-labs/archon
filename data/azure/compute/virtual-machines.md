## Azure Virtual Machines

### Description
Azure Virtual Machines is an on-demand, scalable IaaS compute service that provides Linux and Windows VMs in the cloud with full control over the operating system, networking, and storage configuration. It offers hundreds of VM sizes across general-purpose, compute-optimized, memory-optimized, GPU-enabled, and HPC-optimized families, backed by a 99.99% SLA. Cost optimization is available through Reserved Instances, Spot VMs, Azure Hybrid Benefit (for existing Windows/Linux licenses), and the Azure savings plan for compute.

### Use Cases
* General-purpose web and application servers (e.g., hosting multi-tier web apps behind a load balancer)
* High-performance computing and scientific simulations (e.g., HPC clusters using compute-optimized VM sizes)
* Machine learning training and inference (e.g., GPU-enabled NC/ND-series VMs for deep learning workloads)
* Lift-and-shift of on-premises workloads (e.g., migrating SAP, Oracle, or SQL Server VMs with Azure Migrate)
* Windows and Linux enterprise workloads requiring full OS-level control
